# Информационно-справочная система спортивного клуба

Курсовая работа по дисциплине  
**«Современные технологии программирования»**

## Описание
Данный проект представляет собой информационно-справочную систему
спортивного клуба, реализованную на языке Java.
Система предназначена для хранения и отображения справочной информации
о тренерах и расписании тренировок.

## Функциональные возможности
- Просмотр информации о тренерах
- Просмотр расписания тренировок
- Логическая структура системы (модели и контроллеры)

## Используемые технологии
- Java
- GitHub

## Назначение проекта
Учебный проект, разработанный в рамках курсовой работы.
